Readme File

In order to use the app follow the below mentioned steps: 

1.Install all the dependencies in your local environment so that it doesn't affect your global python interpreter.
	*requirements.txt is provided along with the project files. Use the following code to install the dependencies inside the terminal.
	* pip install -r requirements.txt
2.Enable the local environment by running the Activate powershell file inside env\scripts
3.Run the app.py file.
4.Navigate to the port given in the terminal.

*** You must now be able to access the web-app inside the browser ***
